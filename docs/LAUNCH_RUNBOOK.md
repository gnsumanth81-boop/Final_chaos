# Launch Runbook

## Day 0: Emergency Security

1. Revoke and rotate all exposed keys.
2. Regenerate Telegram bot token.
3. Scope GitHub token to the single repository.
4. Search all workflow exports for secrets.
5. Re-import sanitized workflow only.

## Day 1: Local Build

```powershell
npm test
node scripts/run-brief.js
```

If no OpenRouter key is configured, the system uses deterministic fallback logic. That is expected for a smoke test.

## Day 2: Supabase

1. Create project.
2. Run `supabase/schema.sql`.
3. Add service role key to n8n credentials.
4. Create Supabase insert node for `chaos_signals`.
5. Create one insert per play into `chaos_plays`.

## Day 3: n8n Workflow

Use `n8n/NODE_MAP.md`.

Paste code nodes from:

- `n8n/code/parse-validate-output.js`
- `n8n/code/signal-ledger-payload.js`
- `n8n/code/build-telegram-free.js`
- `n8n/code/build-telegram-pro.js`
- `n8n/code/alert-evaluator.js`

## Day 4: Dashboard

Deploy `public/`.

Required files:

- `public/index.html`
- `public/api/latest.json`
- `public/api/signals.json`

In production, these can be generated by scripts, GitHub Actions, or n8n.

## Day 5: Telegram

Free channel:

- Headline
- Signal
- Fear/BTC
- Safe play
- Link to full brief

Pro channel:

- Full trap
- Full edge
- All plays
- Key levels
- Alerts

## Day 6: Resolver

Run hourly:

1. Query unresolved signals older than 24h.
2. Fetch current BTC/SPX/VIX.
3. Calculate result.
4. Update row.

## Day 7: Soft Launch

Do not market win rate yet.

Say:

"The ledger is live. We are publishing every signal and every resolution."

## Pre-Sell Checklist

- [ ] No hardcoded secrets
- [ ] Latest run succeeds
- [ ] Signal inserted into Supabase
- [ ] Signal visible on dashboard
- [ ] Telegram Free sent
- [ ] Telegram Pro sent
- [ ] Resolver tested
- [ ] No fake performance
- [ ] Disclaimer visible
- [ ] Checkout tested

## First Marketing Claim Allowed

Only after at least 30 resolved signals:

"Public signal ledger now live with X resolved signals."

Do not claim a win rate until the sample size is meaningful.
